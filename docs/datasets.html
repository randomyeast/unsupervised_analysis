lib.util.datasets¶

  • lib.util.datasets.mouse_v1
class lib.util.datasets.core.TrajectoryDataset(data_config, all_states=None)¶

Base class for datasets of fixed-length trajectories.

__getitem__(idx)¶

Return a fixed length trajectory from either training or testing set.

__init__(data_config, all_states=None)¶

Abstract dataset class for fixed-length trajectory datasets. When working with any new type of dataset, create a new subclass of this class.

Parameters:

data_config (dict) –

A dictionary containing the following model attributes:
  • test_proportion: float (optional)

    Proportion of dataset to use for testing.

  • traj_len: int (optional)

    Length of trajectories in frames to use. Not used in this object but it is recommended to use this in the implementation of the subclass.

__len__()¶

Return the length of the dataset.

eval()¶

Set the dataset to evaluation mode.

fit_preprocess()¶

Fit parameters used for preprocessing e.g. singular value decomposition

load_data(data_config)¶

Load the dataset, this must be implemented in the subclass used for training the model. It must return a tensor of shape [num_trajectories, trajectory_length, feature_dim]

train()¶

Set the dataset to training mode.
